\# Activity A17 – Types of Locks



\## 1. Suitcase Lock
(suitcase lock.jpeg)



A suitcase lock is a combination-based lock used to secure luggage. It protects personal belongings during travel.



\---



\## 2. Bicycle Lock

(bicycle lock.jpeg)



A bicycle lock is used to prevent theft by securing the bike to a fixed object. It provides strong physical protection.



\---



\## 3. Zip Lock (Backpack Lock)

(zip lock.jpeg)



A zip lock is used on bags to prevent unauthorised access. It adds an extra layer of security for personal items.



\---



\## 4. Drawer Lock

(drawer lock.jpeg)



A drawer lock is used to secure drawers and protect important items or documents from unauthorised access.

\# Activity A12 – Offline Security Tools



\## 1. House Key

(house key.jpeg)



A house key is a physical security tool used to unlock doors and restrict access to authorised users.



\---



\## 2. Student ID Card

(student id card.jpeg)



A student ID card is used for identity verification and helps confirm that a person is authorised.



\---



\## 3. Sealed Envelope

(sealed envelope.jpeg)



A sealed envelope protects the confidentiality of documents and prevents unauthorised viewing of the contents.



\---



\## 4. Notebook

(notebook.jpeg)



A notebook can store personal or sensitive written information and helps protect privacy when kept securely.



\---



\## 5. USB Drive

(USB drive.jpeg)



A USB drive is an offline storage device used to carry important files without relying on internet access.

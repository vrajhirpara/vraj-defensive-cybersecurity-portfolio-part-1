\# Activity A11 – Access Control Devices



\## 1. Biometric (Fingerprint)

(biometric.jpeg)



Uses fingerprint recognition to verify identity and allow secure access.



\---



\## 2. PIN Keypad

(pin keypad.jpeg)



Requires a code to grant access, ensuring only authorised users can enter.



\---



\## 3. QR Code System

(QR code.jpeg)



Allows access by scanning a code linked to a specific online service or function.



\---



\## 4. NFC (EFTPOS)

(eftpos.jpeg)



Uses contactless payment where users tap their phone or card to complete secure transactions.



\---



\## 5. RFID (Transperth)

(transperth-rfid.jpeg)



Uses a card-based system where users tap to access public transport services.

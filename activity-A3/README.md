\# Activity A3 – Security Concepts Used in My House



\## 1. Security Alarm Keypad

(security alarm keypad.jpeg)



This alarm keypad is used to arm and disarm the home security system. It helps detect unauthorised entry and improves household safety.

\## 2. Window Lock

(window lock.jpeg)



The window lock helps prevent unauthorised access through windows, which are common entry points in a house.



\---



\## 3. Door Lock

(door lock.jpeg)



The door lock is a basic physical security measure that restricts access and protects the home from intrusions.



\---



\## 4. Sliding Door Lock

(sliding door lock.jpeg)



The sliding door lock secures another access point in the house and helps prevent unauthorised entry.



\---
